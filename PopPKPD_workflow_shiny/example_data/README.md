# Example Data

## This folder contains example datasets used in the PopPK / PKPD Workflow Shiny App.

Files
## warfarin_PKS.csv

Population PK dataset for warfarin.
Used for:

-PK structural model development

-Transit absorption comparison

-Diagnostics and VPC

## warfarin_dat.csv

Combined PK/PD dataset.
Used for:

-Sequential PK → PD modeling

-Simultaneous PKPD modeling

-Exposure–response demonstration
